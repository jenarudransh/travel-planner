# 🗺️ India Travel Route Planner: Algorithmic Visualizer

An interactive, canvas-based pathfinding visualizer built with Vanilla JavaScript. This project demonstrates complex data structures and graph algorithms in real-time, calculating the most efficient routes between major Indian cities.

## ✨ Features

* **Algorithm Comparison Engine:** Run paths using both **Dijkstra's Algorithm** (exploratory) and **A* (A-Star) Search** (heuristic-driven) to physically see the difference in "Nodes Explored."
* **Dynamic Roadblocks:** Right-click any road on the map to block it (simulating traffic or accidents). The algorithm instantly recalculates a detour.
* **Interactive Canvas:** Built entirely from scratch using HTML5 `<canvas>`. Features include node hovering, pulsing radar beacons, and a custom map coordinate system.
* **Live Route Traversal:** Once a path is found, a simulated vehicle navigates the glowing route at 60fps.
* **Trip Economics:** Calculates estimated fuel cost, carbon footprint (CO₂), and total amenities (petrol pumps & food) along the chosen route.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Glassmorphism UI)
* **Logic/Engine:** Vanilla JavaScript (ES6+)
* **Data Structures:** Graphs, Priority Queues
* **Algorithms:** Dijkstra's Algorithm, A* Search
